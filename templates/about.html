About TigerTalk

TigerTalk is a discussion platform exclusive to the Princeton University community. TigerTalk has been created as a final project for COS333: Advanced Programming Techniques taught by Brian Kernighan in Spring 2018.

We understand that you might be concerned about privacy. Please be aware that all user identities are completely anonymous to other users. Our dedicated team can see which users are making posts, but we will only access this information if a post is reported. Feel free to reach out to a member of our team if you have any concerns.

We encourage you to provide any feedback that you might have. We are always looking to improve the user experience.

For more information about our project, please check out our design document, which can be found here.
